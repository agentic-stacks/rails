# rails

An [agentic stack](https://github.com/agentic-stacks/agentic-stacks) that teaches AI agents how to operate rails.

## Usage

```bash
agentic-stacks init agentic-stacks/rails my-project
cd my-project
agentic-stacks pull
```

Then start Claude Code — it reads `.stacks/rails/CLAUDE.md` and becomes an expert operator.
